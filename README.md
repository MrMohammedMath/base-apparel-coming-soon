### **Usage**

```bash
  git clone https://github.com/MrMohammedMath/base-apparel-coming-soon.git

  cd base-apparel-coming-soon

  npm install
```

- for running dev server

```bash
npm run start
```

- for build & review

```bash
npm run build && npm run preview
```
